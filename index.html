charset="UTF-8"> Uso das Tags HTML

Tag <header>

A tag <header> é usada para definir o cabeçalho de uma seção ou página. Normalmente, contém o logotipo e o título do site.

Tag <nav>

A tag <nav> é usada para definir uma seção de navegação. Ela geralmente contém links para outras partes do site, como menus de navegação.

Tag <section>

A tag <section> é usada para dividir o conteúdo em seções temáticas. Ela ajuda a organizar o conteúdo da página de forma lógica.

Tag <article>

A tag <article> é usada para marcar conteúdo independente e autocontido, como notícias, postagens de blog ou artigos. Pode ser usado dentro de uma tag <section>.

Tag <footer>

A tag <footer> é usada para definir o rodapé de uma seção ou página. Normalmente, contém informações de contato, direitos autorais e links importantes.
